# Deskripsi
Portal ini merupakan sistem informasi berbasis web yang digunakan untuk mendukung proses pemilihan pegawai teladan di lingkungan BPS Provinsi Sulawesi Utara secara objektif, transparan, dan terstruktur.

# Tujuan
- Mempermudah proses penilaian pegawai.
- Mengurangi proses manual.
- Menyediakan hasil penilaian secara otomatis dan terdokumentasi.

# Teknologi
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
